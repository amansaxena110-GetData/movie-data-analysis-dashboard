# Movie Data Analysis Dashboard

## 📊 Project Overview
This project analyzes movie data to identify trends in releases, awards, and duration using Power BI.

## 🛠 Tools Used
- Power BI
- Excel

## 📌 Key Insights
- Total Movies: 65K
- Total Awards: 438
- Total Nominations: 1038
- Average Duration: 83.89 minutes
- Movies trend analysis by release year
- Genre-wise awards analysis

## 📈 Features
- Interactive dashboard with filters (Year, Genre)
- Visual representation of movie trends
- Awards and nominations breakdown
- Duration analysis
